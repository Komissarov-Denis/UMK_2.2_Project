1.2 Классификация, архитектура, интерфейсы, шины.

Компьютеры могут быть классифицированы по различным признакам. Вот некоторые из наиболее распространенных признаков классификации ПК:

Функциональное назначение:
  • настольные ПК (стандартные, игровые, рабочие);
  • ноутбуки (ультрабуки, игровые, бизнес-ноутбуки);
  • серверы (файловые, веб-серверы и т.д.);
  • планшеты;
  • смартфоны.
Архитектура процессора:
  • x86 (Intel и AMD);
  • ARM (преимущественно используется в мобильных устройствах).
Операционная система:
  • windows;
  • macOS;
  • linux;
  • chrome OS и другие.
Размер и форм-фактор:
  • настольные ПК (стандартные, мини-ПК, микро-ПК);
  • ноутбуки (стандартные, ультрабуки, 2-в-1 устройства);
  • планшеты (стандартные, гибридные).
Назначение использования:
  • для офисной работы;
  • для игр;
  • для мультимедиа и творчества;
  • для разработки программного обеспечения;
  • для научных и инженерных расчетов;
  • для домашнего использования.
Характеристики и возможности:
  • процессор;
  • оперативная память;
  • объем накопителя (жесткий диск/SSD);
  • графический ускоритель;
  • порты и интерфейсы;
  • экран (в случае ноутбуков и планшетов).
