Инструкция пользователя


Для начала работы установка программы не требуется, достаточно скопировать папку с продуктом на компьютер или съемный носитель информации.

Далее, нужно запустить файл index.html через любой браузер, щелкнув по файлу 2 раза правой кнопкой мыши. Открывается начальная страница.


При нажатии на кнопку «Теоретический материал» открывается следующая страница.


Теоретический материал представлен двумя разделами.


При нажатии на один из них, можно выбрать тему и перейти на страницу с информацией по этой теме.


Вернуться к выбору другой темы можно нажав на кнопку внизу «В раздел».


При нажатии на кнопку «Практические занятия» осуществляется переход на страницу выбора между двумя разделами.


Далее можно открыть любое практическое занятие.


Внизу расположено две кнопки:

1. «В раздел» - осуществляется переход к выбору практического занятия;

2. «Видео» - открывает видео к данному занятию. Оно открывается небольшим окном, которое можно увеличить на весь экран при необходимости.


При нажатии на кнопку «Контроль знаний», осуществляется переход на страницу с нумерацией тестов относительно тем.


При нажатии на кнопку «Справка», открывается небольшое окно с информацией про данный программный продукт.


Для перехода на основные страницы сайта, можно воспользоваться кнопкой в верхнем правом углу.


Она открывает всплывающее меню.
